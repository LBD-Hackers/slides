Tech intro

What is Linked Data?

BIM tools have an underlying data model that allow us to capture the world as we see it.

This is the same case for linked data except that the data model is expected to never be complete.

Knowledge Graph

In linked data and the semantic web we use a simple framework to capture this knowledge graph

RDF: Resource Description Framework

Querying the graph is as easy as decribing a path along its edges

SPARQL: SPARQL Protocol and RDF Query Language

The loosely coupled graph can be distributed across the web and so can the underlying schemas

Discoveries:
