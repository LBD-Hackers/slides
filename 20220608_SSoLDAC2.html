ConTech project

It was investigated how boundary conditions for walls can be described in a semantic data model so:

  1. The architect doesn't need to create detailed drawings
  2. The architect can continiously check the design against manufacturer solutions
  3. IKEA like manuals could be machine generated for the specific wall
  4. The wall could potentially be custom made by the manufacturer

A set of rules were defined

A demo model was created
